# Installation

## Package Managers

```bash
npm install iocraft
```

```bash
yarn add iocraft
```

```bash
pnpm add iocraft
```

## Requirements

- Vue ^3.0.0
- Node.js >= 18.0.0

## Optional: Router Integration

```bash
npm install vue-router@^4.0.0
```

## TypeScript

TypeScript is supported out of the box.
